## اسکیما

- **WebSite**: برای معرفی سایت‌ها.
- **WebPage**: برای معرفی صفحات وب.
- **Organization**: برای معرفی یک سازمان، شرکت یا نهاد. (مثلاً یک شرکت بزرگ یا موسسه دولتی)
- **LocalBusiness**: برای معرفی کسب‌وکارهایی که در یک موقعیت جغرافیایی خاص فعالیت دارند.
- **Corporation**: برای معرفی یک شرکت یا موسسه بزرگ.
- **Person**: برای معرفی فردی خاص (مثلاً یک نویسنده، دانشمند یا هنرپیشه).
- **Product**: برای معرفی محصولات فیزیکی یا دیجیتال.
- **Service**: برای معرفی خدماتی که کسب‌وکارها ارائه می‌دهند.
- **Offer**: برای معرفی پیشنهادات فروش یا تخفیف‌ها.
- **Reservation**: برای مدیریت رزروهای خدمات یا محصولات.
- **Menu**: برای معرفی منوهای رستوران‌ها و کافه‌ها.
- **FAQPage**: برای معرفی صفحات پرسش و پاسخ.
- **HowTo**: برای معرفی دستورالعمل‌های قدم به قدم.
- **JobPosting**: برای معرفی فرصت‌های شغلی.
- **Event**: برای معرفی رویدادهای خاص.
- **Recipe**: برای معرفی دستورپخت‌ها.
- **Place**: برای معرفی مکان‌ها (مکان‌های عمومی یا خصوصی).
- **PostalAddress**: برای معرفی آدرس‌ها.
- **Review**: برای معرفی نقد و بررسی‌ها.
- **Rating**: برای معرفی امتیازها.
- **AggregateRating**: برای جمع‌بندی و تحلیل نمرات.
- **MedicalEntity**: برای معرفی هر نوع موجودیت پزشکی، از جمله بیماری‌ها و درمان‌ها.
- **MedicalBusiness**: برای معرفی کسب‌وکارهای مرتبط با سلامت.
- **Drug**: برای معرفی داروها.
- **HealthTopic**: برای معرفی موضوعات بهداشتی.
- **ImageObject**: برای معرفی تصاویر.
- **VideoObject**: برای معرفی ویدیوها.
- **AudioObject**: برای معرفی فایل‌های صوتی.
- **Offer**: برای معرفی پیشنهادات فروش.
- **PriceSpecification**: برای معرفی جزئیات قیمت، مثل واحد قیمت یا تخفیف‌ها.
- **DiscountOffer**: برای معرفی پیشنهادات تخفیف.
- **CreativeWork**: برای معرفی آثار خلاقانه مانند مقالات، کتاب‌ها، فیلم‌ها و موسیقی.
- **Book**: برای معرفی کتاب‌ها.
- **Movie**: برای معرفی فیلم‌ها.
- **Article**: برای معرفی مقالات یا نوشته‌ها.
- **MusicRecording**: برای معرفی آثار موسیقی.
